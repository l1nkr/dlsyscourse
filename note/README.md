### Lecture 7

关于静态图和动态图的区别。
主要区别在于构建计算图的过程中有没有包含计算过程。静态图在构建计算图时，并不会执行实际的计算。但是动态图在构建计算图时，会包含计算过程。所以两者的优势和劣势都比较明显，静态图可以更好进行优化，但是不利于调试，动态图则反之。
